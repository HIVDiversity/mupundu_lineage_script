# Mupundu Script

Tool to create a CSV file with lineages as numbers, from a FASTA file. For example:

| new_names    | lineage_num | lineage |
| ------------ | ----------- | ------- |
| NNNNNNNN_TP2 | 2           | mx3     |
| NNNNNANN_TP2 | 2           | mx3     |
| BNNNNNNN_TP1 | 1           | mx1     |
